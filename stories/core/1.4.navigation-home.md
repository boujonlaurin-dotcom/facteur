### Navigation Structure
```
/ (redirect selon auth)
├── /login
├── /onboarding
├── ShellRoute (bottom nav)
│   ├── /feed (Tab 1)
│   │   └── /feed/content/:id
│   ├── /saved (Tab 2)
│   ├── /sources (Tab 3)
│   │   └── /sources/add
│   └── /settings (Tab 4)
├── /progress (modal)
└── /paywall (modal)
```

### Redirect Logic
1. Non authentifié → `/login`
2. Authentifié + onboarding incomplet → `/onboarding`
3. Authentifié + onboarding complet → `/feed`
