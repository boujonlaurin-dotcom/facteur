# Story 10.14 : Relégation du Feed

## Status: Draft

## Story

**As a** utilisateur,  
**I want** pouvoir accéder au feed classique si je le souhaite,  
**so that** je puisse explorer plus de contenu après avoir complété mon digest.

## Acceptance Criteria

1. **Lien "Explorer plus"** discret en bas du DigestScreen
2. **Pas dans la navigation principale** — pas d'onglet dédié
3. **Design minimaliste** pour ne pas détourner du digest
4. **Accessible aussi depuis ClosureScreen**
5. **Feed inchangé** — même fonctionnalité qu'avant

## Dev Notes

### Widget ExploreMoreLink

```dart
// apps/mobile/lib/features/digest/widgets/explore_more_link.dart

import 'package:flutter/material.dart';
import 'package:go_router/go_router.dart';

class ExploreMoreLink extends StatelessWidget {
  const ExploreMoreLink({super.key});

  @override
  Widget build(BuildContext context) {
    return TextButton.icon(
      onPressed: () => context.push('/feed'),
      icon: Icon(
        Icons.explore_outlined,
        size: 16,
        color: Colors.grey[600],
      ),
      label: Text(
        'Explorer plus',
        style: TextStyle(
          color: Colors.grey[600],
          fontSize: 13,
        ),
      ),
      style: TextButton.styleFrom(
        padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 8),
      ),
    );
  }
}
```
