# BMad Method Workflow

Tu travailles sur le projet **Facteur** en suivant le **BMad Method**.

## Règles critiques

1. **Toujours créer/mettre à jour les stories** dans `docs/stories/` avant/après implémentation
2. **Format de fichier** : `{epic}.{story}.{nom-court}.md`
3. **Lire l'architecture** dans `docs/architecture.md` avant d'implémenter

## Ressources BMAD

Pour accéder aux ressources BMAD, référence ces fichiers :
- Template story : `.bmad-core/templates/story-tmpl.yaml`
- Tâche create-story : `.bmad-core/tasks/create-next-story.md`
- Checklist story : `.bmad-core/checklists/story-draft-checklist.md`
