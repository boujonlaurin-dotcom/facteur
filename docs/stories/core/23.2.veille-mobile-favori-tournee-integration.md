## Vision révisée (PO, 2026-05-19)

L'original PR-3 de Story 23.1 prévoyait un écran feed veille dédié (`/veille`). La vision a pivoté :

- **Entry point** : la page "Mes intérêts" (CTA "Crée ta veille" + favori veille avec badge), **plus de "Configurer ma veille"** dans le settings sheet.
- **Affichage du contenu** : un slot `kind=veille` dans la Tournée du jour (`flux-continu`), avec accent visuel `sectionVeille1/2`.
- **Flow config** : 5 écrans (intro → step1 → step1.5 preset → step2 → step3), step4 frequency drop, **skip** sur step2 + **mode "Configuration avancée"** sur step2/step3.
- **Suppression** : dashboard, deliveries, delivery detail, transitions, step4.
