# Story 12.4 : Aligner ThemeToSourcesMapping avec les slugs ML

> **Epic**: [[epic-12-taxonomy-alignment.md]]
> **Priorité**: P1
> **Estimation**: 1h

---

## Contexte

Le fichier `theme_to_sources_mapping.dart` utilise des slugs de subtopics dans `bySubtopic` qui ne correspondent pas aux `VALID_TOPIC_SLUGS` du pipeline ML. Résultat : certains mappings sont morts (ne matchent jamais).

## Écarts identifiés

| Slug actuel | Slug ML correct | Action |
|-------------|----------------|--------|
| `middle-east` | `middleeast` | Renommer |
| `physics` | `science` | Remapper |
| `biology` | `science` | Remapper (doublon avec physics → fusionner sources) |
| `arts` | `art` | Renommer |
| `french-politics` | `politics` | Renommer |
| `labor` | `work` | Renommer |
| `macroeconomics` | `economy` | Renommer |

## Travail

### 1. Corriger les slugs dans `bySubtopic`

```dart
// Avant
'middle-east': ['Le Collimateur', 'Le Monde Diplomatique'],
'physics': ['ScienceEtonnante', 'Epsiloon'],
'biology': ['Epsiloon', 'The Conversation'],
'arts': ['Le 1 Hebdo'],
'french-politics': ['Le Monde', 'Mediapart', 'Le Figaro'],
'labor': ['Alternatives Économiques'],
'macroeconomics': ['Heu?reka', 'Nouveau Départ'],

// Après
'middleeast': ['Le Collimateur', 'Le Monde Diplomatique'],
'science': ['ScienceEtonnante', 'Epsiloon', 'The Conversation'],  // fusion physics+biology
'art': ['Le 1 Hebdo'],
'politics': ['Le Monde', 'Mediapart', 'Le Figaro'],
'work': ['Alternatives Économiques'],
'economy': ['Heu?reka', 'Nouveau Départ'],
```

### 2. Vérifier qu'il n'y a pas de conflit de clés

Si `politics` existe déjà comme clé dans `bySubtopic`, fusionner les listes de sources.

### 3. Ajouter des entrées sport si pertinent

Si des sources sport existent, ajouter :
```dart
'sport': ['L\'Équipe', ...],
```
