# Story 12.3 : Converger les macro-thèmes UI vers les 9 thèmes backend

> **Epic**: [[epic-12-taxonomy-alignment.md]]
> **Priorité**: P1
> **Estimation**: 2-3h

---

## Contexte

Le fichier `topic_labels.dart` définit 8 macro-thèmes UI pour grouper les 51 topics dans les écrans (MyInterests, TopicExplorer) :
- Tech & Science, Societe, Environnement, Culture, **Lifestyle**, **Business**, International, **Autres**

Ces groupes ne correspondent **pas** aux 9 thèmes backend :
- tech, society, environment, economy, politics, culture, science, international, **sport**

Exemples d'incohérences :
- `travel`, `gastronomy` → UI "Lifestyle" mais backend "culture"
- `startups`, `finance` → UI "Business" mais backend "economy"
- `history`, `philosophy` → UI "Autres" mais backend "culture"
- `sport` → UI "Lifestyle" mais backend "sport" (thème dédié)

## Objectif

Remplacer les 8 groupes UI par les 9 thèmes backend, pour que toute la plateforme parle le même langage taxonomique.

## Travail

### 1. Mettre à jour `_slugToMacroTheme` dans `topic_labels.dart`

Remplacer par un mapping qui suit exactement `TOPIC_TO_THEME` du backend :

| Topic slug | Ancien groupe UI | Nouveau thème |
|------------|-----------------|---------------|
| ai, tech, cybersecurity, gaming, privacy | Tech & Science | **Tech** |
| space, science | Tech & Science | **Science** |
| politics | Societe | **Politique** |
| economy | Societe | **Économie** |
| work, education, health, justice, immigration, inequality, feminism, lgbtq, religion, wellness, family, relationships, factcheck | Societe | **Société** |
| climate, environment, energy, biodiversity, agriculture, food | Environnement | **Environnement** |
| cinema, music, literature, art, media, fashion, design, travel, gastronomy, history, philosophy | Culture + Lifestyle + Autres | **Culture** |
| startups, finance, realestate, entrepreneurship, marketing | Business | **Économie** |
| geopolitics, europe, usa, africa, asia, middleeast | International | **International** |
| sport | Lifestyle | **Sport** |

### 2. Mettre à jour `macroThemeOrder`

```dart
const List<String> macroThemeOrder = [
  'Tech',
  'Science',
  'Société',
  'Politique',
  'Économie',
  'Environnement',
  'Culture',
  'International',
  'Sport',
];
```

### 3. Adapter les labels

Les noms de groupe passent de l'anglais technique (Tech & Science, Societe) au français propre correspondant aux thèmes.

### 4. Vérifier les écrans consommateurs

Rechercher tous les usages de `getTopicMacroTheme()` et `macroThemeOrder` pour s'assurer que les écrans s'adaptent :
- `MyInterestsScreen`
- `TopicExplorerSheet`
- Tout autre écran groupant les topics par macro-thème

## Fichiers à modifier

## Fichiers à vérifier (impact)
