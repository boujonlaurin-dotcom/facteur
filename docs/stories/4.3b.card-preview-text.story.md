# Story 4.3b: Preview Text pour articles sans image

## Status: Done

## Story

**As an** utilisateur,
**I want** voir un court texte de prévisualisation (preview) lorsque l'article n'a pas d'image,
**so that** le feed soit plus engageant et informatif même sans visuels.

## Acceptance Criteria

- [ ] **Data Availability**: La description courte issue du flux RSS doit être renvoyée par l'API dans le feed principal.
- [ ] **UI Fallback**: Si un article n'a pas d'image (`thumbnail_url` est nul ou vide), une zone de texte affichant la description doit apparaître.
- [ ] **Truncation**: La description doit être tronquée à 2 lignes maximum.
- [ ] **Design**: Le texte doit être bien intégré visuellement dans la card, respectant la hiérarchie typographique (moins proéminent que le titre).
- [ ] **Article Detail Behavior**: La description RSS ne doit **PAS** être affichée dans l'écran de détail de l'article. Seul le WebView de l'article complet doit être utilisé.

## Dev Notes

- **Backend**: Modifier `ContentResponse` dans `app/schemas/content.py` pour inclure le champ `description`.
- **Frontend**: Modifier `FeedCard` dans `lib/features/feed/widgets/feed_card.dart`.
- **Performance**: Pas d'impact majeur attendu sur la taille des données du feed car la description est déjà stockée en base.
